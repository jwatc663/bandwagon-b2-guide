## FAQ

**Q：搬瓦工 B2 套餐是官方叫法吗？**

不是。B2 是社区用户给 CN2 GIA-E 第二档套餐（40G 硬盘/2TB 流量）的非正式称呼，官网没有"B2"这个标签。搜这个关键词的朋友，基本上找的是 CN2 GIA-E 40G 那款，年付 $299.99。

**Q：CN2 GIA-E 和 KVM 套餐有什么本质区别？**

线路。KVM 套餐走普通路由，CN2 GIA-E 走电信 CN2 GIA 三网优化线路（全程 CN2 GIA，不是 CN2 GT）。对国内访客来说，CN2 GIA 晚高峰丢包率和延迟都明显更好。建站用途，预算够就选 CN2 GIA-E。

**Q：买了能切换到香港机房吗？**

不能。香港 CN2 GIA 机房是独立套餐，不对 CN2 GIA-E 系列开放迁移。CN2 GIA-E 套餐的可选机房里有十几个，但不包括香港。

**Q：搬瓦工支持退款吗？**

支持。购买后 30 天内不满意可以申请全额退款，包括 IP 被封的情况。

**Q：搬瓦工 B2 套餐适合部署 Dify 或 FastGPT 吗？**

2GB 内存跑 Dify 需要采用存算分离方案（外挂 Supabase 等外部数据库），全功能部署官方推荐 4GB 起。FastGPT 同理。如果只跑 n8n 或者 OpenClaw 之类的轻量级 AI 工具，2GB 够用。
